# openevals 라이브러리로 멀티모달 LLM 평가 가능해졌습니다.

> **출처**: threads @feelfree_ai (Threads)
> **카테고리**: 개발/기술 > LLM 평가 및 라이브러리

## 원문

feelfree_ai

이제 LLM 평가(Eval)할 때 이미지랑 오디오, PDF까지 통째로 던져서 채점할 수 있네요.

랭체인(Langchain)의 오픈소스 LLM 평가 라이브러리인 'openevals'가 대대적으로 업데이트됐습니다. 

가장 눈에 띄는 건 단연 멀티모달 LLM-as-judge 지원입니다. 

텍스트에 국한되지 않고 다양한 포맷을 첨부하면 LLM이 알아서 심판 역할을 해줍니다.

여기에 품질, 안전, 보안 등 실무에서 바로 가져다 쓸 수 있는 20개 이상의 평가용 프롬프트도 함께 추가됐습니다.

LLM 앱 개발하시면서 자체 평가 파이프라인 구축하기 번거로우셨던 분들은, 이 라이브러리 하나로 꽤 많은 리소스와 시간을 아끼실 수 있을 것 같습니다.
