# 로컬에서 LLM 학습을 자동화하는 프로젝트 소개.

> **출처**: threads @feelfree_ai (Threads)
> **카테고리**: 개발/기술 > 인공지능/머신러닝

## 원문

feelfree_ai

OpenAI가 쏘아올린 '10분 안에 LLM 학습하기' 대회, 이걸 복잡한 서버 세팅 없이 로컬에서 허깅페이스(HF) 인프라만으로 끝내버린 프로젝트가 나왔네요.

핵심은 SSH 클러스터 접속이나 동료들과 GPU 눈치싸움 할 필요 없이, 내 PC에서 바로 전체 자동화가 가능하다는 겁니다.

• 연산 스케일링은 Jobs
• 실험 결과 저장은 Buckets
• 지표 로깅은 Trackio

이 모든 걸 HF Hub 위에서 엔드투엔드로 붙였습니다. 

이제 진짜 모델 아이디어만 고민하면 인프라 셋업 없이 알아서 돌아가는 환경이 훌쩍 다가왔네요. 
(심지어 그 아이디어조차 모델이 짜주는 시대지만요)
