# Y Combinator CEO가 Claude Code 스킬셋업을 오픈소스로 공개했습니다.

> **출처**: threads (Threads)
> **카테고리**: 개발/기술 > AI 스킬셋업

## 원문

CHOI (@choi.openai) on Threads

Y Combinator의 CEO 개리 탄(Garry Tan)이 자신이 실무에서 쓰는 Claude Code 전용 스킬 셋업 'gstack'을 오픈소스로 풀었습니다.
CLI 환경에서 웹을 탐색하는 /browse 기능 등 본인의 팁이 담겨 있으며, 텍스트 복사 한 번이면 내 PC에 바로 설치할 수 있습니다.

그는 무작정 코딩을 시키지 않고, 기획을 다듬는 2개의 핵심 스킬을 먼저 활용해 AI가 한 번에 정답을 내놓도록 작업 구조를 짰습니다.

내 업무를 얼마나 정교한 스킬(Skill)과 파이프라인으로 쪼개서 AI에게 쥐여주는지가 정말 중요해지고 있습니다.
