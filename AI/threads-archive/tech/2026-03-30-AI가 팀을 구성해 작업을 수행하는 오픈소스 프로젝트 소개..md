# AI가 팀을 구성해 작업을 수행하는 오픈소스 프로젝트 소개.

> **출처**: threads @feelfree_ai (Threads)
> **카테고리**: 개발/기술 > AI 에이전트

## 원문

feelfree_ai

명령어 하나만 치면 AI가 알아서 서브 에이전트들을 소환해서 팀 단위로 일하는 오픈소스가 나왔습니다.

'ClawTeam'이라는 에이전트 스웜 프레임워크입니다.

복잡한 커스텀 SDK 없이 그냥 CLI 환경에서 바로 돌아가거든요.
리더 AI에게 작업만 던져주면 끝입니다. 

알아서 하위 작업자 AI들을 생성하고, 각자에게 독립된 git 환경(worktree)과 tmux 창을 할당해 줍니다. 

자기들끼리 메시지를 주고받으며 꼬여있는 태스크를 순차적으로 처리하고, 일이 끝나면 알아서 main으로 머지까지 해버립니다.

Claude Code나 Kimi 같은 기존 CLI 에이전트들과 바로 호환된다는 점이 매력적이네요. 

AI 에이전트 하나로도 벅찬데 얘네들이 스스로 팀을 꾸려서 일하는 시대라니, 당장 클론받아서 돌려봐야겠습니다.
