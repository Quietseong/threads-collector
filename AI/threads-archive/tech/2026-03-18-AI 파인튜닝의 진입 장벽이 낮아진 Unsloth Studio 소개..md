# AI 파인튜닝의 진입 장벽이 낮아진 Unsloth Studio 소개.

> **출처**: threads (Threads)
> **카테고리**: 개발/기술 > AI 및 머신러닝

## 원문

CHOI (@choi.openai) on Threads

AI 파인튜닝의 진입 장벽이 사실상 사라졌습니다.

Unsloth AI가 LLM을 쉽게 훈련하고 실행할 수 있는 오픈소스 웹 UI 'Unsloth Studio'를 공개했습니다.
맥이나 윈도우 환경에서 VRAM 사용량을 70%나 줄이면서도 훈련 속도는 2배 높일 수 있는데요.

특히 유저들 사이에서 가장 뜨거운 반응을 얻고 있는 건 '자동 데이터셋 생성' 기능입니다. 
PDF나 CSV 문서만 넣으면 알아서 학습용 데이터를 만들어주어 번거로운 파이프라인 구축 과정이 사라졌습니다.

값비싼 GPU 인프라와 복잡한 데이터 준비 과정에 가로막혀 있던 개인 개발자들에게, 해당 도구는 실질적이고 강력한 무기가 될 것 같습니다.
이제 누구나 내 컴퓨터에서 나만의 맞춤형 모델을 만드는 시대가 성큼 다가왔네요.
