EMNLP 2025 발표된 RAG 프레임워크 LightRAG가 GitHub 트렌딩 중.

도서관 사서가 책 전체를 뒤지는 대신 주제별 연결 지도로 안내하듯, 벡터 검색 대신 지식 그래프로 개념 간 관계를 저장해 복잡한 질문에서도 관련 컨텍스트를 더 정확히 끌어옴.

LangChain·LlamaIndex 같은 기존 RAG 대비 구조가 단순하고 빠른 게 차별점이라고 함.

RAG 파이프라인 직접 구축 중인 바이브코더한테 대안으로 써볼 만한 듯. 논문 기반이라 실제 프로덕션 후기는 더 나와봐야 알 듯.
