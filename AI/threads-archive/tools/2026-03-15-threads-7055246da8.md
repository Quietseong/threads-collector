# HWP 파일을 Markdown으로 변환하는 유용한 도구 소개.

> **출처**: threads (Threads)
> **카테고리**: 도구/리소스 > 문서 변환 도구

## 원문

Threads의 feelfree_ai(@feelfree_ai)님

한국에서 일하다 보면 피할 수 없는 HWP 파일, 이제 명령어 한 줄로 깃이나 노션에 올리기 좋은 마크다운(Markdown)으로 바꿀 수 있습니다.

단순 텍스트 추출을 넘어 복잡한 표나 구조를 최대한 살려주는데, 가장 흥미로운 건 파서 다음에 LLM을 연동해서 사람이 읽기 좋게 자동 포맷팅까지 해준다는 점이네요.

• HWPX / HWP 5.x (2002~2022) 모두 지원
• 로컬 CLI 기반이라 사내망/보안 이슈 해결
• CI/CD나 대량 문서 마이그레이션에 바로 투입 가능
• 필요에 따라 로컬/클라우드 LLM 자유롭게 스위칭

보안 때문에 기존 웹 변환기를 쓰기 찝찝했거나, 사내 레거시 문서를 최신 워크플로우로 옮겨야 하는 분들께 꽤 실용적인 도구가 될 것 같습니다. 

한 번 테스트해 보세요.
