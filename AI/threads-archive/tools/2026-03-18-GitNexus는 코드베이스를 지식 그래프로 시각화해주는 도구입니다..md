# GitNexus는 코드베이스를 지식 그래프로 시각화해주는 도구입니다.

> **출처**: threads @bizmentor_kr (Threads)
> **카테고리**: 도구/리소스 > 코드 시각화 및 탐색 도구

## 원문

BizMentor

브라우저 하나로 코드베이스 전체를 지식 그래프로 변환해주는 도구가 화제입니다. GitNexus는 서버 설치 없이 GitHub 레포나 ZIP 파일만 올리면 의존성, 호출 체인, 실행 흐름을 한눈에 볼 수 있게 시각화해줍니다.

그래프 위에서 AI 에이전트와 직접 대화하며 코드를 탐색할 수 있고, MCP 프로토콜을 통해 외부 AI 도구와도 연결됩니다. 레포 위키를 자동으로 생성하는 기능도 포함되어 있습니다.

처음 투입된 프로젝트나 레거시 코드를 빠르게 파악해야 하는 상황에서 실용적으로 쓸 수 있을 것 같습니다.
