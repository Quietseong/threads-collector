# URL을 Markdown으로 변환해주는 유용한 도구 소개.

> **출처**: threads @iam_mychan (Threads)
> **카테고리**: 도구/리소스 > Markdown 변환 도구

## 원문

마이찬

이런 도구 아직도 모르면 진짜 손해예요. 
링크 하나를 바로 깔끔한 Markdown으로 바꿔주는 툴인데, 생각보다 훨씬 강력합니다. 
특히 로그인 걸린 페이지 긁어와야 할 때 이거 하나면 일이 확 줄어요.

이 서비스 한 줄 요약하면 
“아무 URL이나 Claude에 던지면 읽기 좋은 Markdown으로 정리해주는 도구”예요.

좋았던 건 
X 글, 위챗 글, Feishu/Lark 문서처럼 일반 복붙이 잘 안 되는 페이지도 전용 방식으로 가져온다는 점. 
온라인 프록시가 안 되면 `agent-fetch` 같은 로컬 fallback도 있어서 꽤 든든하고요. 
Feishu 문서는 제목, 리스트, 코드블록, 체크리스트, 이미지까지 Markdown으로 정리돼서 생각보다 완성도가 높아요.
