# Obsidian에서 Claude Code 플러그인을 활용한 메모 관리 방법.

> **출처**: threads @iam_mychan (Threads)
> **카테고리**: 도구/리소스 > Obsidian 플러그인

## 원문

마이찬

이런 플러그인 아직도 모르고 옵시디언 쓰면 손해예요. 
메모 앱이 갑자기 AI 작업실이 됩니다. 
노트 정리만 하던 Vault가, 이제는 직접 읽고 쓰고 실행하는 Claude 작업 공간이 돼요.

Claudian은 Obsidian 안에 Claude Code를 붙여주는 플러그인이에요. 
단순 챗봇이 아니라 파일 읽기/수정, 검색, bash 실행, 멀티스텝 작업까지 가능해요. 
게다가 `@`로 노트 불러오고, 선택한 문장만 바로 고치고, 이미지 분석까지 됩니다. 
데스크톱용이고, Obsidian + Claude Code CLI가 필요해요. 무료 설치는 가능하지만 Claude 구독이나 API는 사실상 필요합니다.
