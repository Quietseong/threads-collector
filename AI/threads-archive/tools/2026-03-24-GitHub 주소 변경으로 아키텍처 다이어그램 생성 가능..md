# GitHub 주소 변경으로 아키텍처 다이어그램 생성 가능.

> **출처**: threads @feelfree_ai (Threads)
> **카테고리**: 도구/리소스 > 코드 시각화 도구

## 원문

feelfree_ai

깃허브(GitHub) 주소창에서 'github'을 'gitdiagram'으로만 바꿔보세요. 복잡한 프로젝트 구조가 즉시 아키텍처 다이어그램으로 변환됩니다. 👀

새로운 오픈소스 분석할 때 진짜 유용한 팁이네요. 
GPT-5.4 mini 기반으로 작동하는데, 다이어그램에서 특정 요소를 클릭하면 해당 소스 파일로 바로 넘어가서 전체적인 코드 흐름 파악하기가 훨씬 수월합니다.

• Mermaid 코드나 PNG 파일로 바로 저장 가능
• 토큰 연결 시 Private 레포지토리도 완벽 지원
• 100% 오픈소스라 직접 호스팅도 가능

처음 보는 레포지토리 구조 파악할 때 들어가는 시간을 엄청 아껴줄 것 같습니다. 당장 써먹어 보세요!
