AI가 짠 코드의 엉성한 디자인에 답답함을 느꼈던 분들에게 매우 반가운 소식이 있습니다.

Claude나 Codex 같은 AI 모델에 직접 연결해 사용하는 디자인 레퍼런스 툴 'Lazyweb'이 완전 무료로 공개되었습니다.

듀오링고 출신 개발자가 만든 해당 프로젝트는 25만 개 이상의 실제 앱과 웹 화면 데이터를 갖춘 해당 서비스는 유료 구독이나 사용량 제한 없이 MCP로 즉시 연동됩니다.

정말 당장은 부족한 부분들이 점점 Skils나 오픈소스로 풀리면서 다양한 분야가 상향 평준화가 되는 것 같습니다.
