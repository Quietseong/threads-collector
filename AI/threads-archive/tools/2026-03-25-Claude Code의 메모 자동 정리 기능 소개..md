# Claude Code의 메모 자동 정리 기능 소개.

> **출처**: threads @iam_mychan (Threads)
> **카테고리**: 도구/리소스 > 메모 관리

## 원문

마이찬

이런 설정 아직도 안 켰으면 손해입니다. 
Claude Code 오래 쓸수록 멍청해진다고 느꼈다면 이유가 있었어요. 
기억은 쌓는 것보다 정리하는 게 더 중요합니다.

Claude Code의 `Auto-dream`은 작업하며 쌓인 메모를 자동으로 정리해주는 기능입니다. 
한마디로, Claude의 기억력을 관리해주는 설정이에요.

유용한 포인트는 이겁니다. 
중복 메모 삭제 
서로 충돌하는 내용 정리 
`어제`, `지난주` 같은 표현을 정확한 날짜로 변환 
쓸모없어진 메모 정리 
주제별 재분류까지 해줍니다.
